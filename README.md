# let.codes eslint

---

## Usage


Install, add config and plugins.

```
yarn add @let-codes/eslint-config --dev
yarn add eslint prettier eslint-config-prettier eslint-plugin-prettier eslint-plugin-react --dev
```

Extend it on your local `.eslintrc` file.

```
{
	"extends": "@let-codes"
}
```
